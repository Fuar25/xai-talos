# Agent Guidelines

## Code Conventions

### Python Code Style

**Indentation**: Use **2 spaces** for indentation in all Python modules.

```python
# Example:
def example_function():
  """Example function with 2-space indentation."""
  if True:
    result = compute_something()
    return result
```

This convention applies to **all Python files** throughout the entire project, including:
- All modules in the `talos/` package
- `setup.py` in the root directory
- Any scripts, tests, or other Python files

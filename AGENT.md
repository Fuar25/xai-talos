# Agent Guidelines

## Code Conventions

### Python Code Style

**Indentation**: Use **2 spaces** for indentation in all Python modules.

```python
# Example:
def example_function():
  """Example function with 2-space indentation."""
  if True:
    result = compute_something()
    return result
```

This convention applies to all Python files in the `talos/` package and throughout the project.
